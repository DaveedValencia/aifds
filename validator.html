AIFDS Validator

The validator scores how ready your JSON is for agent consumption—measuring completeness, freshness, and evidence quality. We’re building it in the open with industry partners.

Request early access Review the doc templates

Scoring pillars

The validator returns independent scores so you know where to invest next.

Understands

Checks template adherence, required field coverage, and schema validation. Surfaces missing context or malformed data structures.

Answers

Benchmarks how well you address known customer questions. Flags absent evidence, stale copy, or broken cross-links.

Acts

Confirms every action resolves predictably with correct HTTP status, security, and post-click experience.
