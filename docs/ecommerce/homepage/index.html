Ecommerce — Homepage

Present a brand-forward overview that primes agents with credibility, merchandising priorities, operations, and the primary paths a shopper can take.

Schema: homepage.schema.json Industry overview: /industries/ecommerce/

Fields

These fields define the minimum contract agents rely on. Optional items let you add richer storytelling without breaking compatibility.

Field Description
template String identifier; always ecommerce_homepage. Required.
version Semantic version of the document contract. Required.
updated_at ISO 8601 timestamp for the last JSON refresh. Required.
site Object containing name, domain, logo, languages, and platform metadata.
about Mission, tagline, and brand story; includes highlights for key differentiators.
credibility Array of trust signals such as certifications, reviews, awards, and partnerships. Each item contains label, value, and optional url.
operations Shipping, returns, fulfillment windows, and support channels. Include contact details.
navigation Primary and secondary nav arrays with label and url, plus featured search shortcuts.
highlights Cards for featured categories, campaigns, or promotions with imagery and supporting copy.
actions Array of user intents (search, shop, support) with type, label, url, and readiness metadata.
