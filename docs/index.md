# AI Feed Discovery Standard (AIFDS)

AIFDS defines how to make business and product data interpretable, verifiable, and trustworthy for AI systems. It provides openly maintained rules, JSON Schemas, and examples so teams can validate structured data with confidence.

## Where to Start

- Read the full specification: [`docs/spec/v1/`](spec/v1/)
- Explore the schema hub with inline examples: [`docs/schemas/`](schemas/)
- Check the roadmap for upcoming profiles: [`docs/roadmap.md`](roadmap.md)

## Feedback & Contributions

Open issues or pull requests on GitHub with proposed rule changes, schema improvements, or example payloads. The standard evolves with the community.
